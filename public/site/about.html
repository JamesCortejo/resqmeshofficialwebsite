About ResQMesh

A local emergency network for connected response.

ResQMesh helps Valencia City prepare verified civilians, mesh devices, rescuers, and administrators for moments when normal communication becomes unreliable.

System overview

Designed for offline access, local alerts, and rescue coordination.

The website is the control point for registration, approvals, devices, rescue teams, distress signals, and deployments. Mesh nodes keep local communication available when internet or mobile networks are weak.

The mobile app connects civilians and rescuers to the system. When a mesh node is online, it syncs important records and activity back to the website.

ResQMesh Connection Flow

How the website, mobile app, mesh devices, and rescue teams work together.

Mobile App Civilians and rescuers
Mesh Device Offline LoRa communication
Admin Website Registration and deployments
Rescue Teams Respond to active emergencies
When internet is unavailable, users connect through mesh devices. When internet returns, devices sync key records and activity back to the website.
